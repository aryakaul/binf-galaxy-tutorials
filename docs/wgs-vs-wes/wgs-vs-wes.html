2.

Now, we will call variants on each of these bam files using FreeBayes. Navigate to the left toolbar and search and select for FreeBayes. Select 'Merge output VCFs' and shift-click to select all bam files corresponding to WGS or WES runs. Set the reference genome as hg38, and in the parameter selection level choose '2. Simple diploid calling with filtering and coverage'. Set 10 as the mic voerage to 'process a site'. Run and repeat for WGS and WES runs. 
See image hint

3.

Now we will use the tool VCF-VCFintersect to compare the two outputs. Find the tool by searching the left toolbar for 'VCF-VCFintersect' and selecting the matching option. Set the first VCF dataset as the FreeBayes output from WES, the second VCF dataset as the FreeBayes output from WGS, and the reference genome as hg38. Leave the rest of the options as their default values and hit 'Execute'.
